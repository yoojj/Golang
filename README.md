# Go
: C 언어에 영향 받은 프로그래밍 언어    
: 익명 함수, 클로저, 일급 함수를 지원하는 명령형 언어         
: 컴파일 언어로 정적 타입 언어이나 동적 타입 언어처럼 사용 가능      


- go programming language
- go framework
    - gin
    - echo -- web
    - buffalo -- web
    - revel -- full stack
    - beego -- full stack
    - gorilla -- web toolkit


```bash
# 설치
$ sudo yum install golang

# 설치 확인
$ go env
```
